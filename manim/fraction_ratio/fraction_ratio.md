封面：用比例来理解分数

场景1：
    1. 当我们理解$\frac{1}{5}$时，我们脑海里想的是：一个蛋糕平均分成了5份，从中拿走1份。这个1份就是蛋糕的$\frac{1}{5}$。
    2. 展示疑问：如何来理解$\frac{6}{5}$，依旧尝试用分蛋糕的方式来理解，突出**没有第6份蛋糕**。
    
场景2：
    1. 引入比例关系。将$\frac{1}{5}$理解为我有1份蛋糕，而你有5份。突出**比例关系**。
    2. 同样的方式理解$\frac{6}{5}$，这时候变成：我有6份蛋糕，而你有5份，完美自洽。

场景3：
    应用，看一道题目：学校某次考试满分的学生有72人，占全校人数的$\frac{3}{100}$，那么全校有多少人？
    分析：72人 比 全校人数 = 3 : 100，那我们列出等式：  $\frac{72}{全校人数}$ = $\frac{3}{100}$
    使用交叉相乘法，我们得到：$72 * 100 = 3 * 全校人数$
    最终得到结果：全校人数=100 * 72 / 3 = 2400

场景4：
    1. 概念：比较的**基准**。
    2. 展示有比较才有高矮，多少。
        2.1 张三身高1.7m, 李四身高1.6m，对于李四来说，张三是高的。此时李四是基准。
        2.2 王五身高1.8m, 对于王五来说，张三是矮的。此时王五是基准。

场景5：
    应用：张三有8块钱，李四比张三多$\frac{1}{4}$, 那么李四有多少钱？
    思考：
        1. 到底李四比张三多多少钱呢？多的钱 比 张三的钱 = 1 : 4 => $\frac{多的钱}{8}$ $\frac{1}{4}$ => 多的钱 = 2
        2. 那么李四有8+2=10块。
    总结：我们发现“基准”总是处于分母位置，是不是一直是这样呢？留大家课后思考~
